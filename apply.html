Iris Gateway Network Application And Requirements


Gateway Application Requirements:

  1. Must be 24/7 capable wihtout user input. (ie. Not run on a ervice that requires the server to be started by a person)
  2. Must have a set of rules
  3. Cannot be a harmful server. (ie. No anarchy, servers used to raid other servers, servers that allow hack/cheats)
  4. Must have a staff team that enforce the rules
