Iris Gateway Network

The Iris Gateway Network is a new Minecraft Bedrock Edition Server advertising Server. Yes that's right, we are a server. We have our own Bedrock Edition Server where your servre will be displayed for others to see and can also be used as a gateway to find other servers.


How do I get my sevrer advertised?

To get your server on our server you need to first fill out the advertisement application and submit it. You then need to join the discord and notify me, Bacon(The Owner), that you have filled out the application and I will go and check it out when I have time.

How will I know if my application was accepted?

I will notify you on discord if your application was accepted or denied. If your application is denied I will notify you along with the reason for denial and I will reset your application so you can try again.

Is there a premium membership?

No. We are a free service but there will be an in-game currency that you can use to buy various perks.

Is there anything else on the server?

YES! We have SkyBlock which is where you will get money to buy the server perks. We also will have some events that will give extra money!

Even though we are not a Mini-Game/Survival/a normal server please remember that we still have rules that must be followed to keep advertisement rights.
